## 方法備註

- 盲版產出：`uv run python scripts/build_review_blind.py --input data/classified_questions_期末.json`（266 題，去答案）
- 8 agent 平行盲答（每 agent 一個獨立 chunk 檔整檔作答，避免 index 切片誤差）；比對腳本驗證 k 集合完全一致後逐題比對
- 與上輪盲審 B（148 題、95.3% 一致）相比，本輪涵蓋全部 266 題（含民權第一批），一致率 98.1%
- 第一輪、第二輪都答錯而「一致」的盲區仍可能存在；本報告只能抓「兩輪不一致」的題
